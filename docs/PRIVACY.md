# WebClip 隐私政策

> 生效日期：2026年3月18日

## 简介

WebClip（"我们"、"本扩展"）是一款 Chrome/Edge 浏览器扩展，旨在帮助用户将网页内容转换为 Markdown 格式。我们高度重视用户隐私，本隐私政策说明了我们如何收集、使用和保护您的信息。

## 信息收集

### 我们收集的信息

**1. 网页内容**
- 仅收集用户主动选择剪辑的网页内容
- 用于转换为 Markdown 格式

**2. 用户设置**
- 主题偏好（浅色/深色/暖纸/GitHub）
- 字体设置（字号、字体类型、行距）
- 存储在浏览器本地

**3. 云笔记配置（可选）**
- 仅在用户主动配置云笔记服务时收集
- 包括：为知笔记等第三方服务的认证信息
- 通过官方 API 进行同步

### 我们不收集的信息

- ❌ 浏览历史记录
- ❌ 未主动剪辑的网页内容
- ❌ 个人身份信息（姓名、邮箱、电话等）
- ❌ 位置信息
- ❌ 设备信息

## 信息使用

### 本地处理
- 所有网页内容的提取和转换在本地完成
- 不上传到我们的服务器

### 数据使用明细（对应 Chrome Web Store 数据使用声明）

根据 Chrome Web Store 开发者政策，我们声明以下数据使用情况：

| 数据类型 | 是否收集 | 用途说明 |
|---------|---------|---------|
| **网络记录** | ✅ 是 | 访问当前活动标签页的 URL 和网页内容，用于提取和转换为 Markdown |
| **用户活动** | ✅ 是 | 记录扩展内部功能使用（如主题切换、设置变更、按钮点击） |
| **网站内容** | ✅ 是 | **核心功能**：提取网页的文字、图片、代码块等内容 |
| 个人身份信息 | ❌ 否 | 不收集姓名、邮箱、电话等 |
| 健康信息 | ❌ 否 | 不收集任何健康相关数据 |
| 财务和付款信息 | ❌ 否 | 无支付功能，不收集财务信息 |
| 身份验证信息 | ❌ 否 | 不收集密码、凭证等 |
| 个人通讯 | ❌ 否 | 不收集邮件、短信、聊天记录等 |
| 位置信息 | ❌ 否 | 不获取 GPS 或地理位置 |

**重要说明**：所有收集的数据仅存储在用户本地浏览器中，我们不会将数据上传到任何服务器或分享给第三方。

### 云同步（用户授权后）
- 仅在用户主动配置后，通过第三方官方 API 同步
- 我们中转但不存储用户的云笔记内容

## 信息共享

我们不会将您的信息出售、交易或转让给任何第三方。

## 数据安全

- 使用 Chrome 扩展标准安全机制
- 敏感数据（如云笔记 Token）加密存储
- 遵循 Chrome Web Store 安全规范

## 用户权利

您有权：
- 随时清除本地存储的数据
- 随时断开云笔记连接
- 卸载扩展以删除所有相关数据

## 第三方服务

本扩展可能连接以下第三方服务：

**为知笔记 (WizNote)**
- 用途：云笔记同步
- 隐私政策：https://www.wiz.cn/
